# Devin AI 工程师范式转移

一个现代化的静态网页,展示Devin AI如何改变软件工程的未来。

## 🌟 特性

- **现代化设计**: 紫色/蓝色渐变主题,玻璃态效果
- **流畅动画**: 滚动触发动画,平滑过渡效果
- **响应式布局**: 完美适配桌面、平板和移动设备
- **交互式轮播**: 自动播放的成功案例展示
- **页面切换动画**: 平滑的导航过渡效果

## 📱 页面内容

1. **Hero Section** - 大标题展示
2. **Introduction** - Devin AI核心特性
3. **Paradigm Shift** - 范式转移时间线
4. **Capabilities** - 六大核心能力
5. **Success Stories** - 交互式案例轮播
6. **Impact** - 影响力数据展示
7. **Future** - 未来展望

## 🚀 在线访问

访问网站: [https://jing-jin.github.io/devin-ai-presentation/](https://jing-jin.github.io/devin-ai-presentation/)

## 💻 本地运行

1. 克隆仓库:
```bash
git clone https://github.com/JiNG-JiN/devin-ai-presentation.git
cd devin-ai-presentation
```

2. 打开网页:
```bash
# 直接打开 index.html
start index.html

# 或查看增强版演示
start demo.html
```

## ✨ 交互功能

### 轮播控制
- **自动播放**: 每5秒自动切换
- **手动控制**: 点击左右箭头按钮
- **点导航**: 点击底部圆点跳转
- **键盘支持**: 使用左右方向键
- **触摸支持**: 移动设备滑动切换
- **暂停功能**: 鼠标悬停暂停自动播放

### 页面导航
- 平滑滚动到各个区域
- 导航链接高亮显示
- 页面切换动画效果

## 🛠️ 技术栈

- **HTML5** - 语义化标记
- **CSS3** - 现代样式,渐变,动画
- **JavaScript (ES6+)** - 交互逻辑
- **Google Fonts** - Inter字体

## 📊 性能优化

- Intersection Observer API实现滚动动画
- 防抖函数优化滚动性能
- CSS transform硬件加速
- 最小化重排和重绘

## 🎨 设计特点

- **渐变色方案**: 紫色/蓝色主题
- **玻璃态效果**: 半透明背景,模糊效果
- **动态背景**: 浮动渐变球体
- **微交互**: 悬停效果,3D卡片倾斜
